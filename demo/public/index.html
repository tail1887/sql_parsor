저장소의 sql_processor_trace 를 호출해, SQL이 Lexer → Parser(AST) → Executor 를 거치는 과정과 실행 결과를 한 화면에서 확인합니다.

주차별로 열기

6주차는 MVP 파이프라인, 7주차는 자동 PK·B+ 인덱스·WHERE id = 중심 예제입니다.

Week 6 INSERT / SELECT 파이프라인 토큰·AST·CSV diff Week 7 학습 가이드 단계별 실습 단계 1~7 · 터미널+SQL · Lexer~결과

실행 방법

  1. 저장소 루트에서 sql_processor_trace 타깃을 빌드합니다.
  2. demo 폴더에서 npm install 후 npm start 를 실행합니다.
  3. 브라우저에서 이 페이지 주소(기본 http://localhost:4010)로 접속합니다.
