SQL Processor 브라우저 데모

SQL을 lexer, parser, executor, CSV 저장 흐름으로 실행하고, trace와 데이터 diff를 브라우저에서 바로 확인할 수 있습니다.

데모 선택

Week 6은 SQL 파이프라인 전체 흐름을 보여주고, Week 7은 자동 ID, 인덱스 기반 WHERE id = ..., B+ 트리 벤치를 다룹니다.

Week 6 INSERT / SELECT 파이프라인 토큰, AST, executor, CSV diff Week 7 자동 ID와 B+ 트리 실험 인덱스 조회 흐름과 벤치 비교

사용 방법

  1. 필요하면 데모 바이너리를 먼저 빌드합니다.
  2. demo 폴더에서 서버를 실행합니다.
  3. http://localhost:4010 을 브라우저에서 엽니다.
